# JARVIS Prototype 🚀

A local-first AI assistant inspired by Ironman JARVIS.

## 🎯 Goal

Build a private offline AI assistant that can:
- Run local LLMs
- Work without internet
- Control applications
- Search local files
- Support voice interaction
- Automate tasks

---

# 🛠️ Tech Stack

- Python
- FastAPI
- Ollama
- Docker
- Git
